# 链表数据结构学习总结

## 最初的问题代码

```C
#include <stdio.h>
#include <stdlib.h>
/*链表节点结构体*/
typedef struct
{
    int val;               // 节点值
    struct ListNode *next; // 指向下一节点的指针
} ListNode;

void printList(ListNode *);

int main()
{
    ListNode *head1;
    ListNode *head2;
    ListNode *head3;
    ListNode *head4;
    int length;
    printf("请输入链表长度：\n");
    scanf("%d\n", &length);
    head1 = (ListNode *)malloc(length * sizeof(ListNode));

    head1->val = 1;
    head1->next = head2;
    head2->val = 2;
    head2->next = head3;
    head3->val = 3;
    head3->next = head4;
    head4->val = 4;
    head4->next = NULL;
    printList(head1);
    free(head1);
    return 0;
}

void printList(ListNode *head)
{
    int i = 1;
    while (head->next != NULL)
    {
        printf("第%d个节点的值为: %d\n", i, head->val);
        i++;
        head = head->next;
    }
}
```

## 问题分析和逐步修改

### 第一步：修正结构体定义

**问题**：在匿名结构体内部使用了 `struct ListNode *next`，但此时 `ListNode` 类型名还没有定义完成。

**解决方案**：
```C
typedef struct ListNode
{
    int val;                    // 节点值
    struct ListNode *next;      // 指向下一节点的指针
} ListNode;
```

**学到的知识**：
- `typedef` 的作用是创建类型别名，让自定义类型像内置类型一样使用
- 结构体自引用必须使用 `struct 结构体名*` 的形式
- `typedef struct ListNode {...} ListNode;` 让我们可以直接写 `ListNode *head` 而不是 `struct ListNode *head`

### 第二步：解决内存分配问题

**问题**：
1. `head2`, `head3`, `head4` 只是指针变量，没有分配内存
2. 访问未分配内存的指针导致程序崩溃

**错误理解**：以为 `malloc(length * sizeof(ListNode))` 会自动为所有指针分配内存

**正确理解**：
- 链表的本质是离散存储，每个节点都要单独分配内存
- 指针只是存储地址的变量，需要指向有效的内存地址

**解决方案**：创建 `createNode` 函数
```C
ListNode * createNode(int val)
{
    ListNode *node = (ListNode *)malloc(sizeof(ListNode));
    if(node == NULL){
        printf("内存分配失败\n");
        return NULL;
    }
    node->val = val;
    node->next = NULL;
    return node;
}
```

### 第三步：修正打印函数

**问题**：`while (head->next != NULL)` 导致最后一个节点打印不出来

**解决方案**：
```C
void printList(ListNode *head)
{
    int i = 1;
    while (head != NULL)  // 修改条件
    {
        printf("第%d个节点的值为: %d\n", i, head->val);
        i++;
        head = head->next;
    }
}
```

### 第四步：实现动态创建链表

**目标**：根据用户输入的长度动态创建链表

**算法思路**：
1. 创建头节点
2. 使用 `current` 指针维护链表尾部
3. 循环创建新节点并连接到链表末尾

```C
head1 = createNode(1);
ListNode *current = head1;
for(int i = 1; i < length; i++)
{
    ListNode *node = createNode(i+1);
    current->next = node;
    current = node;  // 移动尾指针
}
```

### 第五步：正确的内存管理

**问题**：只释放头节点，其他节点内存泄漏

**解决方案**：实现 `freeList` 函数
```C
void freeList(ListNode *head)
{
    while(head != NULL)
    {
        ListNode *temp = head;
        head = head->next;
        free(temp);
    }
}
```

**重要概念**：函数参数是指针的拷贝，修改参数不会影响原指针

## 第三阶段：完整功能实现（增删改查）

### 二级指针的深入理解

在实现链表操作过程中，遇到了二级指针的概念难点：

#### 为什么需要二级指针？
**生活化理解**：
- **普通变量** = 房子
- **一级指针** = 门牌号纸条  
- **二级指针** = 存放纸条的抽屉

```C
int age = 25;        // age住在地址 0x1000
int *p = &age;       // p这张纸条住在地址 0x2000，纸条上写着 0x1000
int **pp = &p;       // pp这个抽屉住在地址 0x3000，里面放着 0x2000
```

#### 链表头插操作的核心问题
```C
// ❌ 一级指针版本（无效）
void addFirst(ListNode* head, int val) {
    ListNode *newNode = createNode(val);
    newNode->next = head;
    head = newNode;  // 只改变了参数拷贝，外部head未变
}

// ✅ 二级指针版本（有效）
void addFirst(ListNode** head, int val) {
    ListNode *newNode = createNode(val);
    newNode->next = *head;
    *head = newNode;  // 修改外部head指针的指向
}
```

**关键理解**：
- **一级指针**：我可以修改你指向的房子里的东西
- **二级指针**：我可以修改你指向哪个房子

### 增加操作实现

#### 1. 头部插入（需要二级指针）
```C
void addFirst(ListNode **head,int val)
{
    ListNode *newNode = createNode(val);
    newNode->next = *head;
    *head = newNode;
}
```

#### 2. 指定位置插入（一级指针足够）
```C
void addIndex(ListNode* head, int index,int val)
{
    ListNode *newNode = createNode(val);
    ListNode *current = head;
    // 找到前驱节点（index-1位置）
    for(int i = 0; i < index-1; i++)
    {
        current = current->next;
    }
    // 连接新节点
    newNode->next = current->next;
    current->next = newNode;
}
```

#### 3. 尾部插入（一级指针足够）
```C
void addLast(ListNode* head,int val)
{
    ListNode *newNode = createNode(val);
    ListNode *current = head;
    // 找到尾节点
    while(current->next != NULL)
    {
        current = current->next;
    }
    current->next = newNode;
}
```

### 删除操作实现

#### 索引计算的重要理解
用户视角 vs 程序索引：
```
用户说的：    "第1个"  "第2个"  "第3个"  "第4个"
程序索引：      0       1       2       3
```

删除第3个节点 = 删除index=2的节点，需要找到index=1作为前驱节点。

#### 1. 删除头节点（需要二级指针）
```C
void removeFirst(ListNode **head)
{
    if(*head == NULL){
        printf("链表为空\n");
        return;
    }
    ListNode *temp = *head;
    *head = (*head)->next;
    free(temp);
}
```

#### 2. 删除指定位置（一级指针足够）
```C
void removeIndex(ListNode *head, int index)
{
    if(head == NULL || index == 0){
        printf("链表为空或尝试删除头节点\n");
        return;
    }
    
    ListNode *current = head;
    // 找到前驱节点
    for(int i = 0; i < index-1; i++)
    {
        current = current->next;
    }
    
    ListNode *nodeToDelete = current->next;
    current->next = nodeToDelete->next;
    free(nodeToDelete);
}
```

#### 3. 删除尾节点（一级指针足够）
```C
void removeLast(ListNode *head)
{
    if(head == NULL){
        printf("链表为空\n");
        return;
    }
    
    if(head->next == NULL) {
        printf("只有一个节点，无法删除，请使用removeFirst函数\n");
        return;
    }
    
    ListNode *current = head;
    // 找到倒数第二个节点
    while(current->next->next != NULL)
    {
        current = current->next;
    }
    
    ListNode *lastNode = current->next;
    current->next = NULL;
    free(lastNode);
}
```

### 查询操作实现

```C
ListNode* get(ListNode *head, int index)
{
    if(head == NULL){
        printf("链表为空\n");
        return NULL;
    }
    ListNode *current = head;
    for(int i = 0; i < index; i++){
        if(current == NULL) {
            printf("索引超出范围\n");
            return NULL;
        }
        current = current->next;
    }
    return current;
}
```

src/
├── linked_list.h          // 头文件：函数声明

### 头文件设计（linked_list.h）
```C
#ifndef LINKED_LIST_H
#define LINKED_LIST_H
#include <stdio.h>

typedef struct ListNode
{
    int val;               
    struct ListNode *next; 
} ListNode;

// 基础操作
void printList(ListNode *);
ListNode * createNode(int val);
void freeList(ListNode *head);

// 增加操作
void addFirst(ListNode **head,int val);
void addIndex(ListNode* head, int index,int val);
void addLast(ListNode* head,int val);

// 删除操作
void removeFirst(ListNode **head);
void removeIndex(ListNode *head,int index);
void removeLast(ListNode *head);

// 查询操作
ListNode* get(ListNode *head, int index);

#endif
```

### 编译问题的解决

#### 头文件包含规则
```C
#include <stdio.h>         // ✅ 系统头文件用<>
#include "linked_list.h"   // ✅ 自定义头文件用""
```

#### 正确的编译命令
```bash
# ❌ 错误：只编译一个文件
gcc .\MyLinkedList.c ; ./a.exe

# ✅ 正确：编译所有源文件
gcc .\MyLinkedList.c .\linkedList_function.c -o test ; .\test.exe
```
