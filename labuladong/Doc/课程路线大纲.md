### Week 5-6: DFS/回溯算法

#### 回溯框架
```cpp
void backtrack(路径, 选择列表) {
    if (满足结束条件) {
        result.add(路径);
        return;
    }
    for (选择 : 选择列表) {
        做选择;
        backtrack(路径, 选择列表);
        撤销选择;
    }
}
```

### Week 7-8: BFS算法

#### BFS框架
```cpp
void BFS(起点) {
    queue.push(起点);
    visited.add(起点);

    while (!queue.empty()) {
        节点 = queue.front();
        queue.pop();

        for (邻居 : 节点的邻居) {
            if (!visited.count(邻居)) {
                queue.push(邻居);
                visited.add(邻居);
            }
        }
    }
}
```
