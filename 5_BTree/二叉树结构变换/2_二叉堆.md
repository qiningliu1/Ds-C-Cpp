## 详细解答和实现

## 二叉堆的性质
- **二叉堆是完全二叉树**：除了最后一层，其他层都是满的，最后一层从左到右连续填充
- **堆序性质**：每个节点的值都大于等于（最大堆）或小于等于（最小堆）其左右子节点的值
- **动态排序**：可以不断添加或删除元素，数据结构会自动调整元素位置保持有序

## 数组表示二叉堆的原理

### 索引对应关系
对于数组中索引为 `i` 的元素：
- **父节点索引**：`(i-1)/2` （整数除法）
- **左子节点索引**：`2*i + 1`
- **右子节点索引**：`2*i + 2`

### 举例说明
数组：`[90, 70, 80, 30, 60, 40, 50]`
```
       90 (索引0)
      /  \
   70(1)  80(2)
   /  \   /  \
30(3) 60(4) 40(5) 50(6)
```

### 原地建堆
- **物理存储**：使用数组，空间复杂度O(1)
- **逻辑结构**：抽象成二叉树
- **核心优势**：不需要额外的TreeNode结构，通过数组索引直接定位父子关系

## 核心操作实现

### 上浮操作(swim)
```cpp
void swim(int node){
    // 如果不是根节点且违反堆序性质，继续上浮
    while(node > 0 && heap[parent(node)] > heap[node]){
        swap(parent(node), node);
        node = parent(node);  // 更新节点位置
    } 
}
```

### 下沉操作(sink)
```cpp
void sink(int node){
    while(left(node) < size){  // 至少有左子节点
        int minChild = left(node);
        
        // 找到更小的子节点
        if(right(node) < size && heap[right(node)] < heap[left(node)]){
            minChild = right(node);
        }
        
        // 如果已经满足堆序性质，停止下沉
        if(heap[node] <= heap[minChild]){
            break;
        }
        
        swap(node, minChild);
        node = minChild;
    }
}
```

## 常见应用-优先级队列

### 完整代码实现
```cpp
#include <iostream>
#include <vector>
using namespace std;

class SimpleMinPQ{
private:
    vector<int> heap;
    int size;

    // 索引计算函数
    static int parent(int node){ return (node-1)/2; }
    static int left(int node){ return node*2 + 1; }
    static int right(int node){ return node*2 + 2; }

    // 上浮操作
    void swim(int node){
        while(node > 0 && heap[parent(node)] > heap[node]){
            swap(parent(node), node);
            node = parent(node);
        } 
    }

    // 下沉操作
    void sink(int node){
        while(left(node) < size){
            int minChild = left(node);
            
            if(right(node) < size && heap[right(node)] < heap[left(node)]){
                minChild = right(node);
            }
            
            if(heap[node] <= heap[minChild]){
                break;
            }
            
            swap(node, minChild);
            node = minChild;
        }
    }

    void swap(int i, int j){
        int temp = heap[i];
        heap[i] = heap[j];
        heap[j] = temp;
    }

public:
    // 构造函数（使用初始化列表）
    SimpleMinPQ(int capacity) : size(0) {
        heap.resize(capacity);
    }

    // const成员函数：只读取，不修改
    int getSize() const { return size; }
    
    int peek() const { 
        if(size == 0) throw runtime_error("队列为空");
        return heap[0]; 
    }
    
    bool isEmpty() const { return size == 0; }

    // 添加元素
    void push(int x){
        if(size >= heap.size()) throw runtime_error("队列已满");
        heap[size] = x;    // 先放在数组末尾
        size++;            // 增加堆大小
        swim(size-1);      // 让新元素上浮到正确位置
    }

    // 删除堆顶元素
    int pop(){
        if(size == 0) throw runtime_error("队列为空");
        int result = heap[0];
        swap(0, size-1);   // 把最后一个元素移到堆顶
        size--;            // 减少堆大小
        if(size > 0) sink(0);  // 让新堆顶元素下沉
        return result;
    }
};
```

### API时间复杂度
- `push(x)`：O(logN) - 需要上浮
- `pop()`：O(logN) - 需要下沉  
- `peek()`：O(1) - 直接返回堆顶
- `getSize()`：O(1) - 直接返回大小

## 堆排序

### 基本原理
把无序数组的所有元素push到堆中，然后依次pop出来，得到有序序列：

```cpp
vector<int> heapSort(vector<int>& arr) {
    vector<int> res(arr.size());
    SimpleMinPQ pq(arr.size());
    
    // 建堆：O(NlogN)
    for (int x : arr)
        pq.push(x);
    
    // 依次弹出：O(NlogN)    
    for (int i = 0; i < arr.size(); i++)
        res[i] = pq.pop();
        
    return res;  // 返回有序数组
}
```

### 时间复杂度分析
- **建堆阶段**：N次push操作，每次O(logN) → O(NlogN)
- **排序阶段**：N次pop操作，每次O(logN) → O(NlogN)  
- **总时间复杂度**：O(NlogN)
- **空间复杂度**：O(N) - 需要额外的堆空间

## 知识点归纳总结

### 核心思想
- **数组抽象成树**：物理存储用数组，逻辑结构是二叉树
- **维护堆序性质**：通过上浮和下沉操作保持父子节点大小关系
- **动态排序**：可以随时添加删除元素，自动维护有序性

### 关键操作
| 操作 | 时间复杂度 | 核心思路 |
|------|------------|----------|
| push | O(logN) | 末尾添加→上浮调整 |
| pop | O(logN) | 删除堆顶→末尾换到堆顶→下沉调整 |
| peek | O(1) | 直接返回数组第一个元素 |

### 实现要点
1. **索引关系**：父节点`(i-1)/2`，左子节点`2i+1`，右子节点`2i+2`
2. **边界检查**：访问数组前先检查索引是否越界
3. **const正确性**：只读函数声明为const，提高代码安全性
4. **内存管理**：使用vector自动管理内存，避免手动delete

### 应用场景
- **优先级队列**：任务调度、事件处理
- **堆排序**：原地排序算法
- **Top K问题**：找最大/最小的K个元素
- **合并K个有序链表**：利用堆的性质高效合并
